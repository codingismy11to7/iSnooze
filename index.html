iTunesAlarm

iTunesAlarm sits in your system tray and lets you schedule times for iTunes to start playing. Spacebar activated snooze, configurable snooze time, gradual volume increasing, multiple alarm schedules, playlist selection, temporary pausing when you reach your computer to give you time to shut it off...why I daresay iTunesAlarm has it all!

Get It

Release 1.2: SetupiTunesAlarm-1.2.exe
Features:

  • Lets you have multiple alarm schedules (have one for the weekend, perhaps, or different times on different days to account for school schedules)
  • Lets you cancel a pending snooze alarm before it goes off (thus stopping the alarm)

Release 1.1: SetupiTunesAlarm-1.1.exe
Features:

  • Automatically opens configure dialog on first run
  • Allows you to select which days the alarm will go off

Release 1.0: SetupiTunesAlarm-1.0.exe
Features:

  • Choose a playlist, optional shuffle
  • Increases volume gradually over a specified time
  • Can minimize iTunes when setting off alarm
  • Can run iTunesAlarm at startup
  • Configurable snooze time
  • Can pause when you hit the mouse or keyboard when the alarm is going off
